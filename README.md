# HangRight

Free interactive wall anchor selector tool. Answer 4 quick questions about your wall type, what you're hanging, and the weight -- get a specific fastener recommendation with buy links.

**Live site:** [https://sillymodes.github.io/hangright](https://sillymodes.github.io/hangright)

## Features

- Step-by-step wizard covering 7 wall types, 9 item categories, and 4 weight classes
- Specific product recommendations with installation steps
- User reviews with star ratings
- Visitor analytics dashboard
- Mobile-friendly, zero-dependency single-page app (vanilla HTML/CSS/JS)
